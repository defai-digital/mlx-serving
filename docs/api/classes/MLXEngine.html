Class MLXEngine

Simplified MLXEngine class for quick prototyping and benchmarking.

This class provides a streamlined API matching mlx-engine's MLXEngine, hiding the complexity of Engine lifecycle management and model loading.

Example: Basic usage

const engine = new MLXEngine("llama-3.1-8b");
await engine.init();

// Simple generate
const text = await engine.generate("Hello!", { max_tokens: 50 });

// Streaming generate
for await (const chunk of engine.generateStream("Hello!", { max_tokens: 50 })) {
  process.stdout.write(chunk.token);
}

// Cleanup
await engine.dispose();

Example: With options

const engine = new MLXEngine({
  model: "llama-3.1-8b",
  quantization: "4bit",
  revision: "main"
});
await engine.init();
